## 🚀 Proyectos destacados

### 🍅 [Pomify](https://pomify.es)
> Aplicación de productividad basada en la técnica Pomodoro

- Interfaz construida con **React + Vite**
- Backend en **Flask (Python)** con autenticación JWT
- Base de datos **PostgreSQL** en producción (Render)
- Sonidos ambientales integrados vía **Freesound API**
- Deploy completo con dominio propio
